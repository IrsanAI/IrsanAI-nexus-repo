# irsanai-nexus-repo — Repository Intelligence Plattform (DE)

**Kurz:** IRSAN AI — Repo Intelligence: Automatisierte, multiperspektivische Analyse von GitHub-Repos mit LLM-Orchestrierung.

## Schnellstart (lokal)
```bash
git clone https://github.com/irsanai/irsanai-nexus-repo
cd irsanai-nexus-repo
chmod +x scripts/setup.sh && ./scripts/setup.sh
source .venv/bin/activate
./scripts/run_local.sh
# Browser: http://localhost:8000
```
