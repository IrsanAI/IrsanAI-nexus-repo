# irsanai-nexus-repo — Repository Intelligence Platform (EN)

**One-liner:** IRSAN AI — Repo Intelligence: Automated, multi-perspective analysis of GitHub repositories with LLM orchestration.

## Quickstart (local)
```bash
git clone https://github.com/irsanai/irsanai-nexus-repo
cd irsanai-nexus-repo
chmod +x scripts/setup.sh && ./scripts/setup.sh
source .venv/bin/activate
./scripts/run_local.sh
# then open http://localhost:8000
```
